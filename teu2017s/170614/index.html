Twitter Widget

Twitter関連の埋め込みWidgetは このページに一覧されている。

TwitterのTweet埋め込みは、ブラウザでTwitterのTweetを表示すれば、メニューから埋め込みタグが取得できる。

Tweetボタン等を置く場合には、事前に こちらのページ を参考に、TwitterのJavaScriptを読み込んでおく必要がある。 headタグ内のBootstrap JavaScript読み込みの後ろあたりに追記しておくと良い。 

<script async src="//platform.twitter.com/widgets.js" charset="utf-8"></script>

上記のscriptが読み込まれていれば、 Tweet Button にあるタグを使うことでTweetボタンなどを挿入することができる。 

<a class="twitter-share-button"
   href="https://twitter.com/intent/tweet?text=Hello%20world"
   data-size="large">
  Tweet</a>

Google Maps

Google Mapsはいくつかの埋め込み方法が提供されているが、最も簡単に使えるのがiframeによる埋め込み機能である。 Google Mapsのページで表示させたい場所を検索したら、共有リンクから地図埋め込みタグを取得できる。

以下、実際に埋め込んだものとなる。

他にも、Google Mapsの「マイプレイス」-> 「マイマップ」で作成した地図は公開設定にすると同様に埋め込むことができるので、地図・地理ベースの情報を効果的に見せたい時には便利である。 埋め込んだ地図の中をJavaScriptで制御したければ、 Maps JavaScript APIを使うことでより凝った見せ方も可能だが、こちらはJavaScript実装の知識がある程度必要でやや難しいので、試してみたい人はサンプルコードを挿入して動かしてみるところから始めると良い。
